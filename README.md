# Rusben CV

Portfolio/CV de Rusben Madrigal — Senior Technical SEO. El diseño original está en [Figma](https://www.figma.com/design/UC8hwv556QDmFADiXC4oEI/Rusben-CV).

## Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Motion** (animaciones)
- **Lucide React** (iconos)

## Cómo ejecutar

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — Servidor de desarrollo
- `npm run build` — Build de producción
- `npm run start` — Servidor de producción (tras `build`)
- `npm run lint` — Linter

## Estructura

- `app/` — Layout, página principal y estilos globales
- `components/` — Secciones y componentes reutilizables (Navigation, Hero, About, etc.)
